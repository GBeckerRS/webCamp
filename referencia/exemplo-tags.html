ADMCode Lab - Web I

Este é um minicurso para qualquer pessoa interessada em aprender o básico sobre programação.

Objetivo

O curso tem como objetivo transmitir de forma prática e eficiente como criar soluções (páginas/aplicações) voltadas para web. Iremos aprender boas práticas e tecnologias atuais, como iniciar uma estrutura básica, como pensar e desenvolver soluções inteligentes para seus clientes. O curso será dividido em pequenos módulos (veja a ementa abaixo), facilitando assim o aprendizado e o desenvolvimento do aluno, visando sempre uma didática simples, concisa e bastante aplicada a prática de trabalho.

O que você vai construir no curso?

No curso utilizaremos as principais ferramentas do mercado para criar a strong{nossa solução}. A nossa solução consiste em reconstruir um layout de uma página de um site conhecido da região.

Para desenvolvermos o novo layout, aprenderemos o básico sobre HTML, CSS e JavaScript.
